## Getting Started

### Prerequisites

- Flutter installed  
- A device or emulator with microphone access  

### Setup

1. **Clone the project:**
   ```bash
   git clone [your-repo-link]
   cd your-project-folder
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Generate necessary files:**
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```

4. **Add `.env` file**

5. **Run the app:**
   ```bash
   flutter run
   ```
